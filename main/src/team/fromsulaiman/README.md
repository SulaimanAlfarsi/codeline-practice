# fromsulaiman - Library CLI Project

A simple Library Management System built with Java.

## Project Structure

```
src/team/fromsulaiman/library-cli/
├── Book.java
├── Library.java
├── Main.java
├── TestRunner.java
├── run.bat
└── .gitignore
```

## How to Run

**Option 1:** Run the batch file
```
Double-click run.bat
```

**Option 2:** Run from terminal
```cmd
cd src/team/fromsulaiman/library-cli
java Main
```

## Features

1. Add Book
2. Show Available Books
3. Borrow Book
4. Return Book
5. Show All Books
6. Show Statistics
7. Remove Book
8. Clear Library
9. Exit
