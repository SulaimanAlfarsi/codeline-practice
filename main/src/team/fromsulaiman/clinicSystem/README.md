# Clinic Appointment CLI

A Java command-line application for managing clinic doctors, patients, and appointments.

## Features

- Add doctors
- Register patients
- Book appointments
- View appointments
- Cancel appointments
- Complete appointments

## OOP Concepts Used

- Encapsulation
- Abstraction
- Inheritance
- Polymorphism
- Interfaces
- Composition
- Single Responsibility Principle

## Run

Compile:

```bash
javac -d out src/main/java/clinic/**/*.java src/main/java/clinic/*.java
